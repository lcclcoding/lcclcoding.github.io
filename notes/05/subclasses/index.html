    Unit 05: Inheritance and Subclasses

    In unit 03, we have seen how we can create complex classes from other smaller classes with composition - modelling a 'has-a' relationship between classes. Now, we will be introducing how we can model 'is-a' relationships using inheritance!

    Square & Rectangle

    Creating a Square

    Suppose now we would like to create a new class called Square. And we decide to implement it this way as:

        class Square:
        def __init__(self, length):
            self._base = length
            self._height = length

        def __str__(self):
            return f"Square with {self.get_base()} length"

        def get_base(self):
            return self._base

        def get_height(self):
            return self._height

        def area(self):
            return self._base * self._height

        def perimeter(self):
            return 2 * (self._base + self._height)

    As you may have noticed, the code look exactly like our trusted Rectangle class from unit 01. If you think about it, a Square is really just a specialized Rectangle.

    In other words, a Square is a Rectangle. Which means we can utilize inheritance to create a subclass.

    Inheriting a Rectangle

    So instead of rewriting our whole code for Rectangle Class as before for our Square class, we can just simply inherit (copy) it from our rectangle class as follow:

    1

        class Square(Rectangle):                    # Over here we tell python we want Square to inherit from the Rectangle class 
        def __init__(self, length):
            super().__init__(length, length)    # super() mean the rectangle class, so what this line does it that is running the __init__ for the Rectangle class that Square inherited from.

        def __str__(self):
            return f"Square with {self.get_base()} length"
    That all, we are done! We will also have access to all the method rectangle in our new Square class.

    class Rectangle:
    def __init__(self, length):
        self._base = length
        self._height = length

    def __str__(self):
        return f"Rectangle with {self.get_base()} length"

    def get_base(self):
        return self._base

    def get_height(self):
        return self._height

    def area(self):
        return self._base * self._height

    def perimeter(self):
        return 2 * (self._base + self._height)

    Now we can say that a Square is a subclass of Rectangle. Or in parent-child relationship, Square is the child of Rectangle. The class diagram as follows:
